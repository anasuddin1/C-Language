# C Language Projects

A collection of beginner C programming projects 
demonstrating core programming concepts.

## Projects

### 🏧 Mini ATM System (atm-final.c)
PIN-based ATM simulator with full transaction functionality.
- Secure PIN login system
- Balance inquiry, deposits, withdrawals
- Receipt printing after transactions
- Menu-driven console interface

### 🎮 Tic-Tac-Toe Game (TicTacToe.c)
Two-player terminal game built in C.
- Dynamic board updates after every move
- Win detection across rows, columns, diagonals
- Draw detection when board is full
- Input validation for invalid moves

### 🔢 Mini Calculator (Mini-Calculator.c)
Basic arithmetic calculator with menu-driven interface.
- Addition, subtraction, multiplication, division
- Simple console UI
